Some notes:
  • Depending on how many scrobbles you have, the app may take a LONG time to load. If you have a lot of scrobbles, I recommend using the "Stop!" button to stop the loading process. This will allow you to see the data that has already been loaded.
  • For its calculations, this app uses the LastFM API, which has some gaps. A lot of songs have no associated Album, Genre, length, etc. To remedy this, press the Deezer button, which will fetch the missing data from the Deezer API.
  • Pressing the "Artist" button will add no extra time to the fetch process. Pressing the "Album" or "Genre" button will roughly double the process. Pressing the "Deezer" button will roughly 4x the process (all very rough estimates depending on your listening habits and Deezer/LastFM's API response times).
  • Genres are based off of LastFM tags. They're community-generated, and very unserious.
